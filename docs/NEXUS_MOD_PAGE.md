## JSON mod managers (v3)

v3 can **export `.json`** camera patches for:

- **[JSON Mod Manager](https://www.nexusmods.com/crimsondesert/mods/113)** (Nexus mod 113)
- **[Crimson Desert Ultimate Mods Manager](https://www.nexusmods.com/crimsondesert/mods/207)** (CDUMM, Nexus mod 207)

Export in UCM → import in the manager you use. JSON **Prepare** requires vanilla `playercamerapreset` on disk (verify game files if you already applied camera mods).
