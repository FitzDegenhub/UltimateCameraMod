> **ปรัชญาการออกแบบ v3: แก้ไขเฉพาะค่า ไม่ฉีดโครงสร้าง**
>
> เวอร์ชันก่อนหน้าจะฉีดบรรทัด XML ใหม่เข้าไปในไฟล์กล้อง (ระดับซูมเพิ่มเติม, โหมดมุมมองบุคคลที่หนึ่งบนม้า, การปรับปรุงกล้องม้าพร้อมระดับซูมเพิ่มเติม) v3 ลบฟีเจอร์เหล่านี้ออกโดยตั้งใจ การฉีดโครงสร้างมีโอกาสสูงที่จะพังหลังอัปเดตเกม และความชอบส่วนตัวสำหรับโหมดกล้องเฉพาะทางนั้นเหมาะสมกว่าที่จะให้ม็อดเฉพาะทางแจกจ่ายผ่านตัวจัดการม็อด ตอนนี้ UCM แก้ไขเฉพาะค่าที่มีอยู่ — จำนวนบรรทัดเท่าเดิม, โครงสร้างองค์ประกอบเดิม, แอตทริบิวต์เดิม ทำให้พรีเซ็ตแชร์ได้อย่างปลอดภัยและทนทานต่อแพตช์เกม

### ตัวแก้ไขสามระดับ (v3)

v3 จัดระบบการแก้ไขเป็นสามแท็บ เพื่อให้คุณสามารถลงลึกได้ตามต้องการ:

| ระดับ | แท็บ | ฟังก์ชัน |
|-------|------|----------|
| 1 | **UCM Quick** | ชั้นเร็ว — ตัวเลื่อนระยะทาง/ความสูง/การเลื่อน, FoV, กล้องกึ่งกลาง, ซูมล็อคเป้า (-60% ถึง +60%), ล็อคเป้าหมุนอัตโนมัติ, ซิงค์ขี่สัตว์, กล้องนิ่ง, แสดงตัวอย่างกล้อง + FoV แบบเรียลไทม์ |
| 2 | **Fine Tune** | การปรับแต่งเชิงลึกที่คัดสรร ส่วนที่ค้นหาได้สำหรับระดับซูมเดินเท้า, ซูมม้า/ขี่สัตว์, FoV ทั่วไป, สัตว์พิเศษและการเดินทาง, ต่อสู้และล็อคเป้า, การปรับกล้องให้นิ่ง และตำแหน่งเล็งและเป้าเล็ง สร้างต่อจาก UCM Quick |
| 3 | **God Mode** | ตัวแก้ไข XML ดิบเต็มรูปแบบ — ทุกพารามิเตอร์ใน DataGrid ที่ค้นหาและกรองได้ จัดกลุ่มตามสถานะกล้อง คอลัมน์เปรียบเทียบค่าเริ่มต้น ค่าศักดิ์สิทธิ์ที่ถูกปกป้อง (สีเขียว) ไม่ถูกเปลี่ยนจากการสร้างใหม่ ตัวกรอง "เฉพาะค่าศักดิ์สิทธิ์" คำอธิบายแอตทริบิวต์ 54 รายการ |

### ระบบพรีเซ็ตแบบไฟล์ (v3)

- **รูปแบบไฟล์ `.ucmpreset`** — รูปแบบเฉพาะที่แชร์ได้สำหรับพรีเซ็ตกล้อง UCM วางลงในโฟลเดอร์พรีเซ็ตใดก็ได้แล้วใช้งานได้เลย
- **ตัวจัดการแถบด้านข้าง** พร้อมส่วนที่จัดกลุ่มแบบพับได้: ค่าเริ่มต้นเกม, พรีเซ็ต UCM, พรีเซ็ตชุมชน, พรีเซ็ตของฉัน, นำเข้า
- **สร้าง / ทำซ้ำ / เปลี่ยนชื่อ / ลบ** จากแถบด้านข้าง
- **ล็อค** พรีเซ็ตเพื่อป้องกันการแก้ไขโดยไม่ตั้งใจ — พรีเซ็ต UCM ล็อคถาวร; พรีเซ็ตผู้ใช้สลับได้ผ่านไอคอนกุญแจ
- **พรีเซ็ตค่าเริ่มต้นแท้** — `playercamerapreset` ดิบที่ถอดรหัสจากไฟล์สำรองเกมของคุณ ไม่มีการแก้ไขใดๆ ตัวเลื่อน Quick จะซิงค์กับค่าพื้นฐานเกมจริง
- **นำเข้า** จาก `.ucmpreset`, XML ดิบ, ไฟล์เก็บ PAZ หรือแพ็กเกจ Mod Manager การนำเข้า `.ucmpreset` จะได้รับการควบคุมตัวเลื่อน UCM เต็มรูปแบบ; การนำเข้า XML/PAZ/Mod Manager ดิบเป็นพรีเซ็ตแบบสแตนด์อโลน (แก้ไขได้เฉพาะใน God Mode ไม่ใช้กฎ UCM) เพื่อรักษาค่าของผู้สร้างม็อดดั้งเดิม
- **บันทึกอัตโนมัติ** — การเปลี่ยนแปลงพรีเซ็ตที่ปลดล็อคจะเขียนกลับไปยังไฟล์พรีเซ็ตโดยอัตโนมัติ (มีการหน่วงเวลา)
- การย้ายพรีเซ็ต `.json` เดิมไปเป็น `.ucmpreset` อัตโนมัติเมื่อเปิดใช้ครั้งแรก

### แคตตาล็อกพรีเซ็ต (v3)

เรียกดูและดาวน์โหลดพรีเซ็ตโดยตรงจาก UCM ดาวน์โหลดด้วยคลิกเดียว ไม่ต้องมีบัญชี

- **พรีเซ็ต UCM** — สไตล์กล้องอย่างเป็นทางการ 7 แบบ (Heroic, Panoramic, Close-Up, Low Rider, Knee Cam, Dirt Cam, Survival) คำจำกัดความโฮสต์บน GitHub, XML เซสชันสร้างในเครื่องจากไฟล์เกมของคุณ + กฎกล้องปัจจุบัน สร้างใหม่อัตโนมัติเมื่อกฎกล้องอัปเดต
- **[พรีเซ็ตชุมชน](https://github.com/FitzDegenhub/UltimateCameraMod/tree/main/community_presets)** — พรีเซ็ตที่ชุมชนร่วมสร้างใน repo หลัก แคตตาล็อกสร้างอัตโนมัติโดย GitHub Actions
- **ปุ่มเรียกดู** บนหัวข้อกลุ่มแถบด้านข้างแต่ละรายการ เปิดตัวเรียกดูแคตตาล็อก
- พรีเซ็ตแต่ละตัวแสดงชื่อ, ผู้สร้าง, คำอธิบาย, แท็ก และลิงก์ไปยังหน้า Nexus ของผู้สร้าง
- **ตรวจจับการอัปเดต** — ไอคอนอัปเดตกระเพื่อมเมื่อมีเวอร์ชันใหม่ในแคตตาล็อก คลิกเพื่อดาวน์โหลดอัปเดต พร้อมตัวเลือกสำรองไปยังพรีเซ็ตของฉัน
- พรีเซ็ตที่ดาวน์โหลดจะปรากฏในแถบด้านข้าง (ล็อคเป็นค่าเริ่มต้น — ทำซ้ำเพื่อแก้ไข)
- **จำกัดขนาดไฟล์ 2MB** และตรวจสอบ JSON เพื่อความปลอดภัย

**ต้องการแชร์พรีเซ็ตของคุณกับชุมชนไหม?** ส่งออกเป็น `.ucmpreset` จาก UCM แล้ว:
- ส่ง [Pull Request](https://github.com/FitzDegenhub/UltimateCameraMod/pulls) เพิ่มพรีเซ็ตของคุณไปยังโฟลเดอร์ `community_presets/`
- หรือส่งไฟล์ `.ucmpreset` ให้ 0xFitz บน Discord/Nexus แล้วเราจะเพิ่มให้

### ส่งออกหลายรูปแบบ (v3)

กล่องโต้ตอบ **ส่งออกเพื่อแชร์** จะส่งออกเซสชันของคุณใน 4 รูปแบบ:

| รูปแบบ | กรณีใช้งาน |
|--------|------------|
| **JSON** (ตัวจัดการม็อด) | แพตช์ไบต์ + `modinfo` สำหรับ **[JSON Mod Manager](https://www.nexusmods.com/crimsondesert/mods/113)** (PhorgeForge) หรือ **[Crimson Desert Ultimate Mods Manager](https://www.nexusmods.com/crimsondesert/mods/207)** (CDUMM) ส่งออกใน UCM แล้วนำเข้าในตัวจัดการที่คุณใช้; ผู้รับไม่จำเป็นต้องมี UCM **เตรียม** จะแสดงเฉพาะเมื่อ `playercamerapreset` สดยังตรงกับสำรองค่าเริ่มต้นของ UCM (ตรวจสอบไฟล์เกมถ้าคุณใช้ม็อดกล้องแล้ว) |
| **XML** | `playercamerapreset.xml` ดิบ สำหรับเครื่องมืออื่นหรือแก้ไขด้วยตนเอง |
| **0.paz** | ไฟล์ที่แพตช์แล้ว พร้อมวางลงในโฟลเดอร์ `0010` ของเกม |
| **.ucmpreset** | พรีเซ็ต UCM เต็มรูปแบบ สำหรับผู้ใช้ UCM คนอื่น |

## วิธีการทำงาน

1. ค้นหาไฟล์เก็บ PAZ ของเกมที่มี `playercamerapreset.xml`
2. สร้างสำเนาสำรองของไฟล์ต้นฉบับ (เพียงครั้งเดียว — ไม่เขียนทับสำรองที่สะอาด)
3. ถอดรหัสรายการในไฟล์เก็บ (ChaCha20 + Jenkins hash key derivation)
4. คลายการบีบอัดด้วย LZ4
5. วิเคราะห์และแก้ไขพารามิเตอร์กล้อง XML ตามการเลือกของคุณ
6. บีบอัดใหม่, เข้ารหัสใหม่ และเขียนรายการที่แก้ไขกลับไปยังไฟล์เก็บ

ไม่มีการฉีด DLL, ไม่แฮกหน่วยความจำ, ไม่ต้องเชื่อมต่ออินเทอร์เน็ต -- แก้ไขไฟล์ข้อมูลล้วนๆ

## คำถามที่พบบ่อย

**จะโดนแบนไหม?**
UCM แก้ไขเฉพาะไฟล์ข้อมูลออฟไลน์ ไม่เข้าถึงหน่วยความจำเกม, ไม่ฉีดโค้ด หรือไม่โต้ตอบกับโปรเซสที่กำลังทำงาน ใช้ตามวิจารณญาณของคุณเองในโหมดออนไลน์/ผู้เล่นหลายคน

**เกมอัปเดตแล้วกล้องกลับเป็นค่าเริ่มต้น**
เป็นเรื่องปกติ — การอัปเดตเกมจะเขียนทับไฟล์ที่แก้ไขแล้ว เปิด UCM ใหม่แล้วคลิกติดตั้ง (หรือส่งออก JSON ใหม่สำหรับ JSON Mod Manager / CDUMM) การตั้งค่าของคุณจะถูกบันทึกอัตโนมัติ

**โปรแกรมป้องกันไวรัสแจ้งเตือน exe**
เป็นการตรวจจับผิดพลาดที่ทราบแล้วกับแอป .NET แบบ self-contained การสแกน VirusTotal สะอาด: [v3.1.2](https://www.virustotal.com/gui/file/7c5ddbfce28cabecb799a00b87ad4c4641c30c9db65cd2560c6a91d578852021) ซอร์สโค้ดเต็มรูปแบบมีให้ตรวจสอบและสร้างด้วยตัวเองที่นี่

**การเลื่อนแนวนอน 0 หมายความว่าอะไร?**
0 = ตำแหน่งกล้องค่าเริ่มต้น (ตัวละครเอียงซ้ายเล็กน้อย) 0.5 = ตัวละครอยู่กึ่งกลางหน้าจอ ค่าลบเลื่อนไปทางซ้ายมากขึ้น ค่าบวกเลื่อนไปทางขวามากขึ้น

**อัปเกรดจากเวอร์ชันก่อนหน้า?**
ผู้ใช้ v3.x: เพียงแทนที่ exe พรีเซ็ตและการตั้งค่าทั้งหมดจะถูกเก็บรักษา ผู้ใช้ v2.x: ลบโฟลเดอร์ UCM เก่า, ตรวจสอบไฟล์เกมบน Steam จากนั้นรัน v3.1 จากโฟลเดอร์ใหม่ ดูรายละเอียดที่[บันทึกการเผยแพร่](https://github.com/FitzDegenhub/UltimateCameraMod/releases/tag/v3.1)
