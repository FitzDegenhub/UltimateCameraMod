# Ultimate Camera Mod - Crimson Desert

Автономный набор инструментов камеры для Crimson Desert. Полный графический интерфейс, предварительный просмотр камеры в реальном времени, три уровня редактирования, пресеты на основе файлов, **экспорт JSON для [JSON Mod Manager](https://www.nexusmods.com/crimsondesert/mods/113)** и **[Crimson Desert Ultimate Mods Manager](https://www.nexusmods.com/crimsondesert/mods/207)** (CDUMM), а также поддержка HUD для сверхшироких мониторов.

## Возможности

### Управление камерой

| Возможность | Подробности |
|-------------|-------------|
| **8 встроенных пресетов** | Panoramic, Heroic, Vanilla, Close-Up, Low Rider, Knee Cam, Dirt Cam, Survival -- с предпросмотром в реальном времени |
| **Пользовательская камера** | Ползунки для дистанции (1.5-12), высоты (-1.6-1.5) и горизонтального смещения (-3-3). Пропорциональное масштабирование сохраняет персонажа в одном и том же положении на экране при любом уровне зума |
| **Поле зрения** | Ванильные 40 градусов до 80 градусов. Универсальное соответствие FoV в состояниях защиты, прицеливания, верховой езды, полета и кинематографических сцен |
| **Центрированная камера** | Персонаж строго по центру в более чем 150 состояниях камеры -- больше никакого смещения камеры через плечо влево |
| **Зум захвата цели** | Ползунок от -60% (приблизить к цели) до +60% (отдалить). Влияет на все состояния захвата цели, защиты и рывка. Работает независимо от Steadycam |
| **Автоповорот при захвате цели** | Отключает автоматический поворот камеры к цели при захвате. Предотвращает резкий поворот камеры к врагам за спиной. Благодарность [@sillib1980](https://github.com/sillib1980) |
| **Синхронизация камеры верховой езды** | Камеры верховой езды соответствуют выбранной вами высоте камеры игрока |
| **Горизонтальное смещение на всех средствах передвижения** | Лошадь, слон, виверна, каноэ, осадная машина и метла -- все учитывают вашу настройку смещения с пропорциональным масштабированием |
| **Согласованность прицеливания способностей** | Фонарь, ослепляющая вспышка, лук и все способности прицеливания/зума/взаимодействия учитывают горизонтальное смещение. Никаких рывков камеры при активации способностей |
| **Сглаживание Steadycam** | Нормализованное время смешивания и колебания скорости в более чем 30 состояниях камеры: покой, ходьба, бег, спринт, бой, защита, рывок/атака, свободное падение, суперпрыжок, притягивание/раскачивание на веревке, отбрасывание, все варианты захвата цели, захват цели верхом, захват цели при воскрешении, враждебность/розыск, осадная машина и все состояния верховой езды. Каждое значение настраивается сообществом через редактор Fine Tune |
| **Sacred God Mode** | Значения, которые вы редактируете в God Mode, навсегда защищены от перестроений Quick/Fine Tune. Зеленые индикаторы показывают, какие значения являются священными. Хранение по пресетам |

> **Философия дизайна v3: только изменение значений, без структурных инъекций.**
>
> Предыдущие версии внедряли новые XML-строки в файл камеры (дополнительные уровни зума, режим от первого лица на лошади, переработка камеры лошади с дополнительными уровнями зума). v3 намеренно удаляет эти функции. Инъекция структуры имеет гораздо более высокую вероятность поломки после обновлений игры, а личные предпочтения для специализированных режимов камеры лучше обслуживаются отдельными модами, распространяемыми через менеджеры модов. UCM теперь изменяет только существующие значения -- то же количество строк, та же структура элементов, те же атрибуты. Это делает пресеты безопаснее для обмена и устойчивее к патчам игры.

### Трехуровневый редактор (v3)

v3 организует редактирование в три вкладки, чтобы вы могли углубляться настолько, насколько захотите:

| Уровень | Вкладка | Что делает |
|---------|---------|------------|
| 1 | **UCM Quick** | Быстрый слой: ползунки дистанции/высоты/смещения, FoV, центрированная камера, зум захвата цели (-60% до +60%), автоповорот при захвате цели, синхронизация верховой езды, steadycam, предпросмотр камеры и FoV в реальном времени |
| 2 | **Fine Tune** | Глубокая настройка. Разделы с поиском для уровней зума пешком, зума лошади/верховой езды, глобального FoV, специальных средств передвижения и перемещения, боя и захвата цели, сглаживания камеры, а также позиции прицеливания и перекрестия. Строится поверх UCM Quick |
| 3 | **God Mode** | Полный редактор XML: каждый параметр в DataGrid с поиском и фильтрами, сгруппированный по состоянию камеры. Колонка сравнения с ванильными значениями. Священные переопределения (зеленые), защищенные от перестроений. Фильтр "Только священные". 54 всплывающие подсказки атрибутов |

### Система пресетов на основе файлов (v3)

- **Формат файла `.ucmpreset`** -- специализированный формат для обмена пресетами камеры UCM. Перетащите в любую папку пресетов, и все заработает
- **Менеджер на боковой панели** со сворачиваемыми группами: Game Default, UCM Presets, Community Presets, My Presets, Imported
- **Создать / Дублировать / Переименовать / Удалить** из боковой панели
- **Блокировка** пресетов для предотвращения случайного редактирования: пресеты UCM заблокированы навсегда; пользовательские пресеты переключаются значком замка
- **Пресет True Vanilla** -- необработанный декодированный `playercamerapreset` из вашей резервной копии игры без каких-либо изменений. Ползунки Quick синхронизированы с реальными базовыми значениями игры
- **Импорт** из `.ucmpreset`, XML, архивов PAZ или пакетов менеджеров модов. Импорт `.ucmpreset` дает полный контроль ползунков UCM; импорт из XML/PAZ/менеджера модов создает автономные пресеты (только редактирование в God Mode, без правил UCM) для сохранения значений оригинального автора мода
- **Автосохранение** -- изменения в разблокированных пресетах автоматически записываются в файл пресета (с задержкой)
- Автоматическая миграция устаревших пресетов `.json` в `.ucmpreset` при первом запуске

### Каталоги пресетов (v3)

Просматривайте и скачивайте пресеты прямо из UCM. Скачивание в один клик, без регистрации.

- **UCM Presets** -- 7 официальных стилей камеры (Heroic, Panoramic, Close-Up, Low Rider, Knee Cam, Dirt Cam, Survival). Определения размещены на GitHub, сессионный XML генерируется локально из ваших файлов игры и текущих правил камеры. Автоматически перегенерируется при обновлении правил камеры
- **[Community presets](https://github.com/FitzDegenhub/UltimateCameraMod/tree/main/community_presets)** -- пресеты от сообщества в основном репозитории, каталог автоматически генерируется GitHub Actions
- **Кнопка "Обзор"** в заголовке каждой группы боковой панели открывает браузер каталога
- Каждый пресет показывает название, автора, описание, теги и ссылку на страницу создателя на Nexus
- **Обнаружение обновлений** -- пульсирующий значок обновления, когда в каталоге доступна более новая версия. Нажмите, чтобы скачать обновление с опциональным резервным копированием в My Presets
- Скачанные пресеты появляются на боковой панели (заблокированы по умолчанию -- дублируйте для редактирования)
- **Ограничение размера файла 2 МБ** и валидация JSON для безопасности

**Хотите поделиться своим пресетом с сообществом?** Экспортируйте как `.ucmpreset` из UCM, а затем:
- Отправьте [Pull Request](https://github.com/FitzDegenhub/UltimateCameraMod/pulls), добавив ваш пресет в папку `community_presets/`
- Или отправьте ваш файл `.ucmpreset` 0xFitz в Discord/Nexus, и мы добавим его за вас

### Мультиформатный экспорт (v3)

Диалог **Экспорт для обмена** выводит вашу сессию в четырех форматах:

| Формат | Применение |
|--------|------------|
| **JSON** (менеджеры модов) | Байтовые патчи + `modinfo` для **[JSON Mod Manager](https://www.nexusmods.com/crimsondesert/mods/113)** (PhorgeForge) или **[Crimson Desert Ultimate Mods Manager](https://www.nexusmods.com/crimsondesert/mods/207)** (CDUMM). Экспортируйте в UCM, импортируйте в ваш менеджер; получателям UCM не нужен. **Подготовить** предлагается только когда активная запись `playercamerapreset` все еще соответствует ванильной резервной копии UCM (проверьте файлы игры, если вы уже применяли моды камеры). |
| **XML** | Необработанный `playercamerapreset.xml` для других инструментов или ручного редактирования |
| **0.paz** | Пропатченный архив, готовый к размещению в папке `0010` игры |
| **.ucmpreset** | Полный пресет UCM для других пользователей UCM |

Включает поля заголовка, версии, автора, URL Nexus и описания для JSON/XML. Показывает количество пропатченных регионов и измененных байтов перед сохранением `.json`.

### Удобство использования

- **Автоматическое определение игры** -- Steam, Epic Games, Xbox / Game Pass
- **Автоматическое резервное копирование** -- ванильная резервная копия перед любой модификацией; восстановление в один клик. Учитывает версии и автоматическая очистка при обновлении
- **Баннер конфигурации установки** -- показывает вашу полную активную конфигурацию (FoV, дистанция, высота, смещение, настройки)
- **Отслеживание патчей игры** -- отслеживает метаданные установки после применения; предупреждает, когда игра могла обновиться, чтобы вы могли повторно экспортировать
- **Предпросмотр камеры и FoV в реальном времени** -- вид сверху с учетом дистанции, горизонтальным смещением и конусом поля зрения
- **Уведомления об обновлениях** -- проверяет релизы на GitHub при запуске
- **Ярлык папки игры** -- открывает директорию игры из заголовка
- **Идентификация в панели задач Windows** -- правильная группировка значков и значок в заголовке окна через shell property store
- **Сохранение настроек** -- все выбранные параметры запоминаются между сеансами
- **Изменяемый размер окна** -- размер сохраняется между сеансами
- **Портативное приложение** -- один `.exe`, установщик не требуется

### Философия

> **Никто еще не довел камеру Crimson Desert до совершенства -- и в этом весь смысл.**
>
> В ванильной игре более 150 состояний камеры, каждое с десятками параметров. Ни один разработчик не может настроить все это под каждый стиль игры и экран. Именно поэтому существует UCM -- не чтобы сказать вам, какая камера идеальна, а чтобы дать вам инструменты найти ее самостоятельно и поделиться с другими.
>
> Каждую настройку, которую вы изменяете, можно экспортировать и поделиться. Исправление автоповорота при захвате цели, устранившее рывки камеры в бою, было обнаружено одним участником сообщества, экспериментировавшим в God Mode. Именно такая тонкая настройка силами сообщества -- то, для чего создан этот инструмент.

### Обмен пресетами

Экспортируйте вашу настройку камеры как файл `.ucmpreset` и поделитесь с другими. Импортируйте пресеты из каталога сообщества, Nexus Mods или от других игроков. UCM также экспортирует в JSON (для [JSON Mod Manager](https://www.nexusmods.com/crimsondesert/mods/113) и [CDUMM](https://www.nexusmods.com/crimsondesert/mods/207)), необработанный XML и прямую установку PAZ.

## Как это работает

1. Находит PAZ-архив игры, содержащий `playercamerapreset.xml`
2. Создает резервную копию оригинального файла (только один раз -- никогда не перезаписывает чистую резервную копию)
3. Расшифровывает запись архива (ChaCha20 + деривация ключа Jenkins hash)
4. Распаковывает через LZ4
5. Разбирает и изменяет XML-параметры камеры на основе ваших выборов
6. Повторно сжимает, повторно шифрует и записывает измененную запись обратно в архив

Без инъекции DLL, без модификации памяти, без подключения к интернету -- чистая модификация файлов данных.

## Сборка из исходного кода

Требуется [.NET 6 SDK](https://dotnet.microsoft.com/download/dotnet/6.0) (или новее). Windows x64.

### v3 (рекомендуется)

Закройте все запущенные экземпляры перед сборкой -- шаг копирования exe завершится ошибкой, если файл заблокирован.

```powershell
Stop-Process -Name "UltimateCameraMod.V3" -Force -ErrorAction SilentlyContinue
dotnet build "src/UltimateCameraMod.V3/UltimateCameraMod.V3.csproj" -c Release
Start-Process "src/UltimateCameraMod.V3/bin/Release/net6.0-windows/UltimateCameraMod.V3.exe"
```

### Зависимости (NuGet -- восстанавливаются автоматически)

## Часто задаваемые вопросы

**Меня забанят за это?**
UCM модифицирует только офлайн-файлы данных. Он не затрагивает память игры, не внедряет код и не взаимодействует с запущенными процессами. Используйте на свой страх и риск в онлайн/мультиплеерных режимах.

**Игра обновилась, и моя камера вернулась к ванильной.**
Это нормально -- обновления игры перезаписывают измененные файлы. Откройте UCM заново и нажмите "Установить" (или повторно экспортируйте JSON для JSON Mod Manager / CDUMM). Ваши настройки сохраняются автоматически.

**Мой антивирус пометил exe.**
Известный ложноположительный результат для автономных приложений .NET. Проверка VirusTotal чистая: [v3.2](https://www.virustotal.com/gui/file-analysis/ZWMzZGM4MGM3ZWFlZTY5MTFmZDYwYzNkODFlZGM4Mjg6MTc3NTkxMzY4Mg==). Полный исходный код доступен здесь для проверки и самостоятельной сборки.

**Что означает горизонтальное смещение 0?**
0 = ванильная позиция камеры (персонаж немного левее). 0.5 = персонаж по центру экрана. Отрицательные значения сдвигают еще левее, положительные -- еще правее.

**Обновляетесь с предыдущей версии?**
Пользователи v3.x: просто замените exe -- все пресеты и настройки сохранятся. Пользователи v2.x: удалите старую папку UCM, проверьте файлы игры в Steam, затем запустите v3.1 из новой папки. Подробные инструкции в [примечаниях к релизу](https://github.com/FitzDegenhub/UltimateCameraMod/releases/tag/v3.1).
